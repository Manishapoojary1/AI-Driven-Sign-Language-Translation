# AI-Driven Sign Language Translation

## Project Overview
This project translates sign language gestures into text using Artificial Intelligence
and Deep Learning. The system works on uploaded images and supports alphabets and
common gestures such as Hello, Hi, Bye, and Good Morning.

## Features
- Image-based sign language recognition
- Alphabet and gesture detection
- Deep Learning using CNN
- Simple and user-friendly implementation

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## How to Run the Project
```bash
pip install -r requirements.txt
python app.py

## Dataset
Dataset is collected from Kaggle and custom gesture images.
